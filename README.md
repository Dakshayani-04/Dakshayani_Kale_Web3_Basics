# Dakshayani_Kale_Web3_Basics
Web3 and Blockchain Basics: Setup Wallet and Explore Dapps


## Project Overview
This project is designed to help users understand the basics of **blockchain technology**, the concept of **Web3**, and how to interact with **decentralized applications (DApps)**. It involves setting up a **MetaMask wallet**, obtaining **testnet Ethereum (ETH)**, and interacting with a **DApp (Uniswap)** on the **Sepolia Test Network** to perform a token swap. The goal is to provide hands-on experience in interacting with a blockchain network in a secure, test environment.

## Objective
The objective of this task is to:
- Set up a **MetaMask wallet** to interact with the blockchain.
- Obtain **testnet ETH** from a faucet.
- Interact with a **DApp (Uniswap)** to perform a test transaction (token swap).
- Document the process with **screenshots** and a **reflection** on the experience.

## Prerequisites
### Prerequisites
- **MetaMask** browser extension installed (compatible with Chrome, Firefox, or Brave).
- Basic understanding of **blockchain technology**, **Ethereum**, and **Web3 concepts**.
- An internet connection to access faucets and DApps.
- Familiarity with **block explorers** like Etherscan.

## Setup Instructions

### 1. Install MetaMask
- Download and install the **MetaMask** extension from [https://metamask.io](https://metamask.io).
- Create a new wallet and store your **seed phrase** securely offline.
- Configure MetaMask to use the **Sepolia Test Network** in the settings under **Networks**.

### 2. Obtain Testnet ETH
- Go to a **Sepolia Faucet** 
- Copy your **MetaMask wallet address** and paste it into the faucet to request testnet ETH.

### 3. Connect to Uniswap
- Go to **[Uniswap](https://app.uniswap.org)** (Testnet version).
- Connect your **MetaMask wallet** to Uniswap.

### 4. Perform a Token Swap
- On Uniswap, swap **testnet ETH** for **USDC** or **DAI**.
- Approve the transaction in MetaMask and confirm the swap.

### 5. Verify Transaction
- Visit **Etherscan** and search for your wallet address to view the transaction hash.

## Challenges Encountered
### Challenges Encountered
- **MetaMask not syncing** with the Sepolia network: Resolved by clearing the account activity data and resetting the wallet.
- **DApp connection issues**: Fixed by re-adding the Sepolia test network manually in MetaMask.
- **Faucet delays**: Switched to a different faucet after waiting for too long.

These challenges gave me hands-on experience in troubleshooting Web3 tools and understanding how decentralized systems work.
